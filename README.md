# 🛍️ GenZ Mart

Modern online shopping website with smooth animations and shopping cart.

## ✨ Features

- Glassmorphism UI with blur effects
- Shopping cart with localStorage
- Category filters (Shirt, tshirt, Accessories, Sneakers)
- Smooth animations & hover effects
- Fully responsive (mobile + desktop)
- Real product images
